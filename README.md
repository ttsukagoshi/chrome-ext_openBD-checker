# openBD情報チェッカー（Google Chrome拡張機能）
ウェブブラウザ（Chrome）内で選択したISBNコードに基づいて、コンテクストメニュー（右クリックメニュー）から[openBD](https://openbd.jp/)に登録されている情報を参照できるようにする拡張機能。

## 使用
fugafuga
- ハイフン（-）が入ったISBN文字列でも検索可能。
